# Road-Accident-Dashboard

## Project Overview

This project focuses on analyzing road accident data to identify key patterns, trends, and insights related to accidents. The dataset was first explored and cleaned using Microsoft Excel, and then an interactive dashboard was created using Power BI to visualize important metrics.
The goal of this project is to understand accident trends, identify high-risk areas, and present insights through clear and interactive visualizations.

## Tools & Technologies Used

- Microsoft Excel – Data cleaning, preprocessing, and basic analysis
- Power BI – Data modeling and dashboard creation
- Data Visualization – Charts, KPIs, and interactive filters

## Dashboard Features

- The Power BI dashboard includes the following visualizations:
- Total Accidents KPI – Shows the total number of accidents.
- Total Casualties Analysis – Displays injuries or deaths caused by accidents.
- Accidents by Road Type – Identifies which road types have more accidents.
- Accidents by Weather Conditions – Shows how weather affects accident rates.
- Monthly / Yearly Trend Analysis – Tracks accident patterns over time.
- Location-Based Analysis – Highlights accident distribution across different areas.
- Interactive filters allow users to explore the data dynamically.

 ## Key Insights

- Certain road types and weather conditions have higher accident rates.
- Accident trends change depending on time and location.
- Visualization helps quickly identify high-risk situations and possible improvement areas.

## Project Objective

The main objective of this project is to transform raw accident data into meaningful insights using data analysis and visualization techniques.

- This dashboard can help:
- Understand accident trends
- Support safety awareness
- Assist in decision-making for traffic management

## Conclusion

This project demonstrates how Excel and Power BI can be used together to clean, analyze, and visualize data effectively. The interactive dashboard makes it easier to understand accident patterns and communicate insights clearly.
